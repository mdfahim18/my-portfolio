## Run Locally

Clone the project

```bash
  git clone https://github.com/mdfahim18/my-portfolio.git
```

Go to the project directory

```bash
  cd my-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
